Chrono

 

Chrono V1.0 allows you to chronometer up to 4 drivers with the same device and at the same time. You can see for each driver the last lap time, the best lap time and the average lap time.


All stopwatches continue to run even if you receive a call or if you switch to another application. When you come back to Chrono, chronometer continue to run as if you did not quit.

Chrono V1.1 is available on the App Store.

This new version of Chrono corrects a bug when user click multiple time on the start stop button. This new release also add the following feature:

  1. Buttons have now different color. When stopwatch is running button is red otherwise it is green.

  2. It is now possible to choose the precision of stopwatches. (0,1s or 0,01s or 0,001s)
